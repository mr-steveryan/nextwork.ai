### Exploring Claude's capabilities

When I asked Claude what it can do, it told me
Hi! I'm Claude Code, Anthropic's CLI assistant for software engineering. Here's what I can help with:

Code tasks
- Write, edit, debug, and refactor code in any language
- Explain how code works
- Review diffs and pull requests
- Run tests and fix failures

Project work
- Navigate and explore codebases
- Create and manage files
- Run shell commands, build tools, package managers
- Work with git (commits, branches, PRs via gh)

Research & web
- Search the web and fetch page content
- Look up documentation, APIs, libraries

Specialized skills (type / to trigger)
- /code-review — review your current diff for bugs and improvements
- /run — launch and drive your app
- /verify — confirm a change works as intended
- /security-review — security audit of pending changes
- /simplify — clean up changed code
- /schedule — set up recurring automated tasks

Integrations available
- Google Drive, Gmail, Google Calendar
- Canva
